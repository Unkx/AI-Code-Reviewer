# CodeLens — AI Code Review

Paste code, select language, get senior-level review feedback (bugs, security, quality, suggestions) plus an auto-generated fix proposal. Powered by Llama 3.3 via Groq.

## Stack
- Next.js 16 (App Router, Turbopack)
- Tailwind CSS v4
- TypeScript
- Groq SDK (Llama 3.3 70B)

## Surface
Single-page tool. No auth, no routing. One textarea input, one results view.

## Users
Developers pasting code snippets for quick review. Desktop-primary, keyboard-heavy, in-flow usage.
